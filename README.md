# Tic-Tac-Toe (Vanilla JS)

A clean, modular Tic-Tac-Toe game built with Vanilla JavaScript, focusing on the Module Pattern and Factory Functions.

## Key Features
- **Module Pattern (IIFE):** Encapsulated game logic to prevent global scope pollution.
- **Factory Functions:** Player object creation.
- **Responsive UI:** Dynamic DOM manipulation with grid.
- **Game Logic:** Win detection, draw handling, and turn-switching.

## Tech Stack
- HTML, CSS, JavaScript
